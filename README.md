# Estudo de Java

## Descrição
Projeto destinado ao estudo de Java: do básico ao avançado. 

**Objetivo**: aprofundar os conceitos da linguagem e praticar recursos fundamentais e avançados.

## Objetivos
- Estudar sintaxe e fundamentos (tipos, controle, coleções).
- Aprofundar em orientação a objetos (classe, herança, interfaces, encapsulamento).
- Explorar APIs da plataforma (Streams, IO, Concurrency, JDBC).
- Praticar padrões de projeto, testes e boas práticas.
- Implementar projetos e exercícios para fixação.

## Estrutura do projeto
- `src/main/java` — código fonte.
- `src/test/java` — testes unitários.
- `build/` ou `target/` — saída de build (dependendo da ferramenta).
- `docs/` — anotações e material de estudo.
